Face Recognition Application: Photo Finder
implementation of LBPH algorithm

Finding your photos in a random photo album, using your one reference picture(1x1) ...

############################ PROGRAM FLOW ###########################

1) At the GUI. fill first the username.
2) select the reference photo by clicking the menu > open reference photo
3) click select album folder to browse the random photo album
4) start photo finder

the program will create new folder in the directory. The new folder will be label by the username of the user. if photos matched, the matched photos will be copied and paste into this new folder.
